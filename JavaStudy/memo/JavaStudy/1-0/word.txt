IT用語

■IT
IT コンピュータとネットワークを駆使した技術
ICT　ほとんどITと同意語。個人から情報発信をするという意味を強調する。
IoT　物にセンサーを付け、センサーが取得した情報を活用すること。アレクサ、スマートロック

■OS
OS　ソフトウェアのこと。winかmicか、androidかアップルか
Windows　マイクロソフト社製のOS
macOS　アップル社製のOS
UNIX　もっとも古いOS。カーナビとかに使われている。
Linux　OS。UNIXをもとに作られた。androidのOSに使われている。

■言語・環境
マックアップ　HTMLやCSSがマックアップ言語。文章構造やレイアウト、装飾などを指定していくこと。
フロントエンド　JavaScriptなどの言語を使用してコーディングすること。
サーバーサイド　サーバー（作り手）側のこと。JavaSE基本機能、JavaEEWEB開発に使用。

■サーバーソフト
サーバーソフト　サーバー機能を提供するソフトウェア。
Apache　WEBサーバー。半数で使用されている。
nginx　速度が速いWebサーバー。大量処理も得意。使用数第２位。
IIS　マイクロソフト製のWEBサーバー。

■SQL
SQL　データベース言語。データやデータベースを制御する言語。
CRUD　システムの主要機能　生成、読み取り、更新、消去ができるか分析
トランザクション　複数の処理を１つにまとめたもの　失敗（ロールバック）or成功（コミット）のどちらか
排他制御　複数のＰＣからアクセスがあるとき、１つのＰＣがアクセスしているときは他はアクセスできない。
楽観的　編集はできるが取得時とデータが変わっていると更新できない。悲観的　自分が触っているときほかのアクセスをロックする。
実行計画　どうしたら最短でSQLを実行できるかの方法。
チューニング　特定の測定可能かつ達成可能な目標に合うように向上させるプロセス。
